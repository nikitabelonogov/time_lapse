# time_lapse
Time lapse photography with a webcam.

##Usage
```
app.py [-h] [-p CAMERA_PORT] [-c CODEC] [-l TIME_LIMIT]
              [-t OUTPUT_TIME] [-f FPS] [-i INTERVAL]

optional arguments:
  -h, --help            show this help message and exit
  -p CAMERA_PORT, --camera_port CAMERA_PORT
                        The index of the camera.
  -c CODEC, --codec CODEC
                        4 bit codec
  -l TIME_LIMIT, --time_limit TIME_LIMIT
                        Set a time limit in seconds.
  -t OUTPUT_TIME, --output_time OUTPUT_TIME
                        Set a result video time in seconds.
  -f FPS, --fps FPS     Frames per second.
  -i INTERVAL, --interval INTERVAL
                        Time in seconds between shoots.
```
